Detailed Description

A generic, type-safe data container block wrapper utilizing std::any.

Serves as a lightweight polymorphic data attachment layer. It allows high-level framework components (such as asynchronous sockets or compiler-generated coroutine promises) to store, transport, and retrieve arbitrary user context or state data without modifying underlying class hierarchies.
