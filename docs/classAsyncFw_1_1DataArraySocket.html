Detailed Description

An asynchronous socket class for transmitting data arrays (DataArray) with TLS support. Manages high-level data packet processing, read/write buffer boundaries, timeout intervals, keep-alive monitoring, and integration with TLS layers.

Examples
DataArrayTcp/main.cpp, and LogTcp/main.cpp.
