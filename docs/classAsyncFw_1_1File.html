Detailed Description

Provides basic file I/O operations.

A wrapper over the standard std::fstream. The class object is non-copyable and non-movable.

Examples
FileSystemWatcher/main.cpp.

◆ fstream()

std::fstream & File::fstream ( )

Provides direct access to the underlying std::fstream object.

Warning
If the file is opened in any write or modification mode (out or app), calling this method invalidates the internal file size cache (sets size() to std::numeric_limits<std::size_t>::max()).
For read-only modes, the cache remains valid and untouched.
Returns
Reference to the std::fstream instance.
