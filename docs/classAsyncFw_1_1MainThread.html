Detailed Description

A static control interface managing the application's primary execution thread (Main Thread).

MainThread initializes the master Event Loop on the application's boot thread (typically main()). It handles core process-wide shutdown interceptors and manages the primary event loop execution frame.

Example:

#include <core/LogStream.h>
#include <main/MainThread.h>
int main(int argc, char *argv[]) {
AsyncFw::Thread::current()->invoke([]() { AsyncFw::MainThread::exit(123); });
logDebug() << "Start Application";
int ret = AsyncFw::MainThread::exec();
logDebug() << "End Application" << ret;
return ret;
}
