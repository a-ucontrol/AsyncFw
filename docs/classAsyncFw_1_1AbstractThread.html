Detailed Description

The AbstractThread class provides the base functionality for thread management.

Provides core primitives for thread lifecycle management, naming, and scheduling asynchronous tasks via a task queue.

Examples
Coroutine/main.cpp, FunctionConnector/main.cpp, ThreadInvoke/main.cpp, and ThreadPool/main.cpp.
