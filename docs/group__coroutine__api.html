CoroutineAwait< void > AsyncFw::coTimeout ( int ms)

Non-blocking coroutine sleep for the specified timeout interval.

Parameters
msTimeout interval in milliseconds.
Returns
CoroutineAwait object to be used with co_await.

Helper function to pause the current coroutine execution on the spot without creating or managing an explicit AsyncFw::Timer object instance.

Example:

co_await AsyncFw::coTimeout(500); // Sleep for 500 milliseconds without a Timer object
