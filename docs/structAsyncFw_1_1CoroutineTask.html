Detailed Description

A class representing a coroutine task that can be suspended and resumed.

It manages the C++20 coroutine state, holds the underlying handle, and orchestrates the lifecycle via its internal promise_type.

Examples
Coroutine/main.cpp, CoroutineInvoke/main.cpp, CoroutineWait/main.cpp, Socket/main.cpp, and Timer/main.cpp.
