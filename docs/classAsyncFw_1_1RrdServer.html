Detailed Description

A specialized network server component that exposes Round-Robin Databases (Rrd) via TCP.

Binds an instance of DataArrayTcpServer to a collection of Rrd database tracks (metrics, logs, etc.). It automatically listens for incoming client data synchronization queries, extracts historical slices from the ring buffers, and transmits them back to the connected sockets.

Examples
LogTcp/main.cpp.

Extracts a specific historical data slice from an Rrd archive and transmits it to a client socket.

This method acts as the core response handler. When a client requests telemetry or logs, this method reads records starting from a specific timeline index/timestamp up to the requested size, pack into a DataArray frame, and sends it asynchronously.

Parameters
socketPointer to the destination client socket context.
indexThe starting timeline primary index or timestamp boundary to slice from.
sizeMaximum number of data items or chunk size bounds to extract from the target Rrd.
piThe network packet transaction identifier to match the response with the request.
