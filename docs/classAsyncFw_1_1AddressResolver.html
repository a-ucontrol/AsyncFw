Detailed Description

Provides asynchronous DNS resolution using c-ares.

AddressResolver encapsulates low-level POSIX address resolution abstractions. It handles asynchronous domain name resolution (DNS lookups) with support for protocol family selection (IPv4 vs IPv6) and custom per-request timeouts.

Example with FunctionConnector:

AsyncFw::AddressResolver addressInfo2;
addressInfo2.completed.connect([](int, const std::vector<std::string> &list) {
if (list.empty()) {
logError("Resolve error");
} else {
for (const std::string _s : list) logNotice() << _s;
}
});
addressInfo2.resolve("github.com");

Example with CoroutineAwait:

AsyncFw::AddressResolver addressInfo;
AsyncFw::CoroutineHandle h = co_await addressInfo.coResolve("github.com");
AsyncFw::AddressResolver::Result list = h.promise().data<AsyncFw::AddressResolver::Result>();
if (list.empty()) {
logError("Resolve error");
co_return;
}
for (const std::string _s : list) logNotice() << _s;
