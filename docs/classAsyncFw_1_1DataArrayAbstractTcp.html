Detailed Description

Abstract base class implementing a multi-threaded TCP pool for socket and packet management.

Serves as the foundation for both DataArrayTcpServer and DataArrayTcpClient. It manages internal worker threads, applies uniform memory/buffer constraints, handles packet routing, and controls encryption policy rules.
