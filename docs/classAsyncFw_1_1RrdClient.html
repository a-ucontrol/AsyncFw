Detailed Description

A specialized network client class for synchronizing remote Round-Robin Databases (Rrd).

Manages an underlying DataArraySocket to establish communication with an RrdServer. It coordinates a group of local Rrd circular databases, tracks their latest record indexes, and handles secure data ingestion over TCP with optional TLS support.
